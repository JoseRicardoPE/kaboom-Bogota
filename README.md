# Kaboom Bogotá 🎯

Proyecto web desarrollado con **Angular 21** para un centro de entretenimiento en Bogotá.

## 🚀 Tecnologías

* Angular 21
* ng-bootstrap
* ngx-toastr
* Bootstrap 5
* SCSS
* FontAwesome

## 📸 Features

* Hero dinámico con video
* Componentes reutilizables
* Notificaciones con Toastr
* Diseño responsive
* Navegación con Router

## 📦 Instalación

```bash
npm install
ng serve
```

## 🛠️ Desarrollo

Servidor local en:

```
http://localhost:4200
```
